# リリースノート

## Version 2.0

### バージョンの新機能（日本語）

対応エリアを全国10エリアに拡大しました。北海道・東北・東京・中部・北陸・関西・中国・四国・九州・沖縄の電力需給予報を確認できます。また、iOS 26に対応しました。

### What's New (English)

Now covers all 10 regional power areas across Japan — Hokkaido, Tohoku, Tokyo, Chubu, Hokuriku, Kansai, Chugoku, Shikoku, Kyushu, and Okinawa. Also updated for iOS 26.
